Mini E-Commerce Website using AWS

##  Project Overview
This is a mini e-commerce project built using AWS services.  
Users can place an order from a static website, and the backend sends an email notification using AWS SNS.

---

##  Technologies Used
- HTML, JavaScript
- AWS S3 (Static Website Hosting)
- AWS EC2 (Backend Server)
- AWS SNS (Email Notifications)
- Python (HTTP Server)
- Linux (Amazon Linux 2)

---

##  Architecture
S3 Website → EC2 Backend API → SNS Email Notification

---

##  Features
- Static e-commerce website
- Place Order button
- Backend API using Python
- Email notification on order placement
- Deployed fully on AWS
